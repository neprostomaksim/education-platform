# AI Learning — Платформа для изучения ИИ 🚀

Интерактивная образовательная платформа для обучения работе с искусственным интеллектом, промпт-инжинирингу, ИИ-агентам и вайб-кодингу.

---

## 🛠️ Технологический стек

- **Frontend:** Next.js 16 (App Router, Turbopack), React 19, TypeScript
- **Стилизация:** TailwindCSS v4 (инлайновая тема через CSS-переменные в `src/app/globals.css`)
- **Backend / СУБД:** Supabase (PostgreSQL, GoTrue Auth, Row-Level Security)
- **Offline / PWA:** Service Worker (`sw.js`), кэширование курсов, уроков и прогресса в `localStorage`
- **Рендеринг контента:** `react-markdown` с поддержкой таблиц, списков, схем Mermaid и кастомных промпт-карточек

---

## 📂 Структура проекта

### Маршрутизация (Next.js App Router)
- `(auth)` — Вход `/login`, регистрация `/register`, обработчик авторизации `/callback`.
- `(dashboard)` — Основной интерфейс:
  - `/dashboard` — Выбор курсов и прогресс.
  - `/courses/[courseId]` — Программа выбранного курса, разбитая по модулям и тематическим блокам.
  - `/lessons/[lessonId]` — Чтение урока: сайдбар навигации по всему курсу, markdown-контент, фильтрация по ролям.
  - `/prompts` — Статическая библиотека готовых системных промптов с рубрикатором и ролями.
- `(admin)` — Административная панель:
  - `/admin/users` — Список пользователей, ручное создание аккаунтов (обход лимитов SMTP).
- `/pending` — Страница ожидания активации аккаунта администратором.

### Ключевые компоненты логики
- **`UserProvider`** (`src/hooks/use-user.ts`) — Глобальный React Context. Загружает профиль пользователя, обрабатывает оффлайн-кэш и ревалидирует сессию при смене вкладок.
- **`useCourses`** (`src/hooks/use-courses.ts`) — Загружает опубликованные курсы, модули, уроки и прогресс с автоматическим оффлайн-кэшированием в `lms-courses-cache`.
- **`filterContentBySpecialty`** (`src/lib/specialties.ts`) — Динамический парсинг HTML-комментариев вида `<!-- SPEC:role -->` для показа специализированного контента разным профессиям.

## 🏆 Что уже сделано (История изменений)

### 1. Инфраструктура и авторизация
- Настроен Supabase Auth, созданы триггеры для автоматического создания профилей (`profiles`) при регистрации.
- Реализована ролевая модель (`admin` / `student`).
- Внедрен экран ожидания одобрения аккаунта `/pending` с автоматическим редиректом для неактивированных студентов.
- Исправлено зависание интерфейса при фокусе вкладки (оптимизировано получение профиля).

### 2. Административный функционал
- Создан API-клиент Supabase с использованием Service Role Key для обхода ограничений безопасности.
- Добавлена страница управления пользователями: админ видит email, ФИО и статус одобрения.
- Реализована функция ручного добавления студентов без подтверждения по почте (через API Route с проверкой токена авторизации).

### 3. Образовательный контент и курсы
- **Основной курс «Нейросети»**: Полная программа от промпт-инжиниринга до разработки ИИ-агентов. Модули сгруппированы по высокоуровневым блокам.
- **Специализированный курс «ИИ для руководителей бизнеса»**: 7 модулей (17 уроков), ориентированных на CEO и собственников бизнеса (SWOT-анализ, OKR, аудит финансов, HR, продуктовая стратегия).
- **Модули по визуальному контенту**: В оба курса интегрированы уроки по генерации графики, разбору промптов для Nano Banana, Midjourney и DALL-E.

### 4. Дизайн и UX (Библиотека промптов & Интерфейс)
- **Интерфейс библиотек промптов**: Добавлена боковая панель с категориями промптов, счётчиками количества промптов в реальном времени.
- **Улучшено чтение контента**: Контейнеры с промптами больше не обрезают текст (убран фиксированный `max-height` и скролл). Добавлена анимация при копировании.
- **Масштабируемые схемы**: В уроках изображения схем и графиков автоматически растягиваются на 140% ширины контейнера на десктопах, делая мелкий шрифт читаемым.

### 5. Объединение программ (Интеграция ИИ-Агентов)
- **Слияние уроков**: Уроки с практическими промптами для бизнеса (Блоки 2 и 3 курса руководителей) успешно перенесены в тему **«ИИ-Агенты»** основного курса «Нейросети».
- **Новое структурирование**: В теме «ИИ-Агенты» уроки сгруппированы по 7 блокам:
  1. *Основы ИИ-агентов* (теория, архитектура ReAct)
  2. *Стратегия и планирование*
  3. *Финансы и операционка*
  4. *Управление командой и HR*
  5. *Маркетинг и продажи*
  6. *Коммуникация и переговоры*
  7. *Документы и регламенты*
- При этом данные в курсе для руководителей остались нетронутыми.
