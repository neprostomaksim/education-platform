# 🧰 Полка инструментов аудитора — «бери и пользуйся»

> Библиотека готовых инструментов воркшопа «ИИ для аудиторов».
> Часть отрабатывается в зале (навык), часть выдаётся обзором в финале за минуту
> каждый (инструмент). Все макросы проверены на учебных данных.

---

## Как читать таблицу

- **Тип** — Gem (сохранённый промпт в Gemini) или Макрос (VBA в Excel).
- **Зал / Полка** — отрабатываем руками на занятии или выдаём обзором «разберёте после».
- **Учебный файл** — на чём тренироваться (у макросов внутри есть лист «Ключ_аномалий» / «Ключ_расхождений» для проверки).

## Как завести макрос (одинаково для всех)

1. Открыть учебный файл → `Alt+F11` (редактор VBA).
2. File → Import File → выбрать нужный `.bas` (или вставить код в новый модуль).
3. Закрыть редактор → `Alt+F8` → выбрать процедуру → выполнить.
4. Настройки (пороги, праздники) правятся на автосоздаваемых листах.

| Макрос | Что запускать | Что настроить |
|---|---|---|
| JET | `ПрогнатьВсеТесты` (или отдельный `ТестNN`) | лист `JET_настройки`: порог ТЕ, праздники |
| Сверка ОСВ | `ОСВ_Проверка` | ничего (допуск 0,01 в коде) |
| Сверка списков | `Сверка_Списков` | `ДОПУСК` в коде (напр. 1 руб.) |
| Выборка | `Выборка` → метод 1/2/3 | порог и N спросит сам |

## Как завести Gem

1. Десктоп → gemini.google.com → **Gems Manager → New Gem**.
2. Вставить промпт в инструкцию Gem, дать имя.
3. При необходимости — приложить файлы-знания (Памятка, Нормы языка).
4. Сохранить. Дальше просто кидаешь в диалог входные данные.

---

## Три честных правила (проговорить в зале)

1. **Агент оформляет черновик — человек проверяет.** Ни один агент не выносит
   итоговое суждение и не выдумывает НПА.
2. **Числа существенности — формула, не ИИ.** Агенты их не трогают.
3. **Данные обезличиваем перед облаком.** Свободный текст (содержание,
   субконто) проверяем глазами — маскировка ячеек его не ловит.

## Что живёт где

- **На LMS:** все промпты, все `.bas`, учебные файлы, эти три правила, чек-листы
  входа (VPN, аккаунт, New Gem).
- **В зале:** демонстрация + практика по инструментам №1, 3, 4, 5, 6.
- **Мостик к «Дню 2»:** макросы №8–10 показываем обзором → «вживую разберём на
  следующей сессии: сверки, выборка, аналитика ОСВ».
